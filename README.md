## ⚙️ Run Locally

Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

Navigate to the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```
